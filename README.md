# Giovanni Lauffer — Portfolio

Static portfolio site for Giovanni Lauffer, 3D Artist.

## Development

This is a static HTML site deployed on Vercel.

- Edit `public/index.html` for site changes
- Media assets live in `public/wp-content/uploads/`
- Push to `main` to auto-deploy to production

## Deployment

Hosted on Vercel. Auto-deploys on push to GitHub.
